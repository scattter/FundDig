搭建项目描述:

用户在页面上需要输入计划名称, 然后成功创建一个表格, 该表格会被同时保存在后端数据库中, 入库信息包括创建时间, 表格名称等
你认为必要的数据. 
我需要你帮我在当前目录下分别创建front和backend两个文件夹分别保存前后端项目, 期望前端使用rsbuild + react + ts + biome,
后端使用nest + PostgreSQL实现, 如果后端有推荐的日志等系统也帮我一起添加进去. 

前端帮我优化下样式, 添加shadcn和tailwind, scss; 后端帮我优化日志在控制台的输出样式, 便于查看, 同时增加启动时数据库成功连接的信息.
